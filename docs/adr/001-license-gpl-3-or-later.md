# ADR 001: GPL-3.0-or-later License

## Status

Accepted

## Decision

900Word is licensed under GPL-3.0-or-later.

## Context

The product goal is a free, community-owned word processor. Copyleft licensing keeps derived versions free for users and contributors.

## Consequences

- Dependencies, fonts, dictionaries, and future plugins must be GPL-compatible.
- The repo maintains `THIRD_PARTY_LICENSES.md` and `deny.toml`.
- This intentionally differs from 900Invoice's Apache-2.0 model.
