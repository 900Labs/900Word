# Code of Conduct

900Word follows the Contributor Covenant Code of Conduct.

## Our Standards

Participants are expected to:

- Use welcoming and inclusive language.
- Respect differing viewpoints and experiences.
- Accept constructive feedback gracefully.
- Focus on what is best for the community and the users this project serves.

Unacceptable behavior includes harassment, discriminatory language, personal attacks, publishing private information, and other conduct that makes participation unsafe.

## Enforcement

Project maintainers may remove, edit, or reject contributions that do not align with this Code of Conduct. Serious or repeated violations may result in temporary or permanent participation limits.

Report concerns to `opensource@900labs.com`.
